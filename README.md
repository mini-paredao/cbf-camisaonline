# Projetor — Deploy na Vercel

## Como subir
1. Suba esses arquivos no GitHub
2. Conecte o repositório na Vercel
3. A Vercel detecta automaticamente o `vercel.json`
4. Build Command: `npm run build` (já configurado)
5. Output Directory: `.vercel/output` (já configurado)
6. Variável de ambiente: `NITRO_PRESET=vercel` (já no vercel.json)

Pronto. Sem mais tela branca.
